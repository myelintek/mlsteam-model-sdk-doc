Bases: BuiltInPredictorBase

PyTorch built-in multi-class classifier

It supports making (batched) prediction from a Tortch tensor by a single model.

__init__(_submodels: list, submodel: str, classes: Optional[list] = None, batch=True, output_class_dim: Optional[int] = None, **_)

Constructor

Parameters:

  • _submodels – names of all submodels

  • submodel – name of submodel to use

  • classes – a list of class names. The predictor will return class names when classes is provided.

  • batch – enable batch prediction

  • output_class_dim – class probability dimension in the output tensor. It defaults to 0 when batch is disabled and 1 otherwise.
